# Forge Session Solution

## Overview

This creates a complete, production-ready single-page FIFA World Cup 2026 fan application built with React. It includes a hero banner with countdown timer and animated particles, a sticky navigation bar, a filterable match schedule with expandable match summaries, a players spotlight grid with stats cards, a tournament leaderboard with animated progress bars and a podium visualization, a dark-themed news feed with featured articles, and a fully branded footer — all styled in official FIFA colors (deep navy, vibrant red, and metallic gold) with responsive layouts for mobile, tablet, and desktop.

## Task Description

> create a sample one page application that will show me the Fifa World cup 2026 schedule its matches, Est timings, players to look out for in each match with a photo logo, top players till date in the tournament so far, match summary, latest news related to Fifa world cup. Add branding to the site to make it look like an official fifa world cup application
