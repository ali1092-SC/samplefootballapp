# Forge Session Solution

## Overview

Here's a summary of what was built: A fully featured, single-page FIFA World Cup 2026™ fan application built with React + Vite and Tailwind CSS, featuring an animated hero section, filterable match schedule with EST timings and expandable "Players to Watch" cards with photos, a top players leaderboard with medal tiers and performance ratings, tabbed match summaries with goal timelines and stat bars, a news feed with category filtering and modal article overlays, and a branded header/footer — all styled with the official FIFA navy, gold, and red color palette using custom Tailwind tokens and Bebas Neue typography to create an authentic FIFA visual identity.

## Task Description

> create a sample one page application that will show me the Fifa World cup 2026 schedule its matches, Est timings, players to look out for in each match with a photo logo, top players till date in the tournament so far, match summary, latest news related to Fifa world cup. Add branding to the site to make it look like an official fifa world cup application
