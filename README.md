# Project

The deliverable is a single `index.html` file containing the complete FIFA World Cup 2026 fan showcase — all HTML structure, inline CSS, and vanilla JavaScript with zero external dependencies beyond Google Fonts. It includes the hand-coded inline SVG logo at three sizes, a canvas football animation with pentagon patches, a live countdown, seamlessly looping ticker, IntersectionObserver-driven count-up stats and scroll animations, six interactive sections (venues, groups, players, timeline, predictor, footer), full ARIA accessibility, responsive breakpoints down to 375px, and `prefers-reduced-motion` guards throughout.
