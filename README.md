# Project

This is a fully branded, single-page FIFA World Cup 2026™ fan application built in React with no external UI libraries — pure CSS custom properties, animations, and component-level styling. The app features six main sections: an animated hero banner with particle canvas, tabbed match schedule with group/knockout filtering, player watch cards with scout reports, a gold/silver/bronze leaderboard with stat bars, detailed match summary reports with timeline events and possession stats, and a news feed with live ticker — all wrapped in the official FIFA navy/gold/red visual identity with smooth-scroll navigation and a fully responsive layout.
